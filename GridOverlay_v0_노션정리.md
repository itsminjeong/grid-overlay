# Grid Overlay v0

## 1. 제품 개요

### 목적

화면에 표시된 그래프의 특정 영역 위에 반투명 그리드를 겹쳐 표시하여, 데이터의 높이와 위치를 사람이 육안으로 비교할 수 있도록 돕는 Windows용 보조 도구입니다.

### 핵심 원칙

- 화면을 캡처하거나 이미지로 저장하지 않습니다.
- 화면 내용과 사용자 입력을 외부로 전송하지 않습니다.
- 기존 브라우저, 문서, 그래프 화면 위에 그리드만 표시합니다.
- 그래프 값을 자동으로 인식하거나 계산하지 않습니다.
- 현재 v0는 Windows 10/11을 대상으로 합니다.

---

## 2. v0 주요 기능

### 영역 선택

- `F8`을 누르면 그리드를 표시할 영역을 선택할 수 있습니다.
- 마우스로 어느 방향이든 드래그할 수 있습니다.
- 드래그 중 선택 영역의 가로·세로 픽셀 크기를 표시합니다.
- 작은 영역도 선택할 수 있으며 별도의 최소 크기 제한은 없습니다.
- 선택 중 `Esc`를 누르면 선택을 취소합니다.

### 그리드 자동 생성

- 선택이 끝나면 해당 영역 안에만 그리드를 표시합니다.
- 선택 영역의 가로와 세로가 각각 약 10칸이 되도록 초기 간격을 자동으로 설정합니다.
- 선택 영역 밖의 축, 범례, 제목 및 설명은 계속 볼 수 있습니다.
- 선택 영역 밖에는 약한 음영을 표시해 그래프 영역을 구분합니다.

### 그리드 간격 조절

- 일반 휠: X/Y 간격을 함께 2px씩 조절
- `Ctrl + 휠`: X 간격만 0.2px씩 조절
- `Shift + 휠`: Y 간격만 0.2px씩 조절
- `Ctrl + Shift + 휠`: X/Y 간격을 함께 0.2px씩 조절
- 설정창에서 X/Y 간격을 숫자로 직접 입력할 수도 있습니다.
- 그래프 영역에서 휠이 인식되지 않는 환경에서는 설정창의 `여기에 마우스를 올리고 휠` 영역을 이용할 수 있습니다.

### 그리드 위치 조절

- 방향키: 그리드 시작 위치를 1px 이동
- `Shift + 방향키`: 그리드 시작 위치를 5px 이동
- 선택 영역 자체는 유지하고 내부 그리드의 정렬 위치만 변경합니다.

### 표시 설정

- 선 색상 변경
- 선 투명도 변경
- X/Y 간격 직접 입력
- 주선 표시 여부 설정
- 주선 간격 설정
- 기본적으로 5줄마다 더 굵고 진한 주선을 표시합니다.

### 설정창

- 설정창은 그래프를 가리지 않도록 최초에 선택 영역 밖에 배치됩니다.
- 상단의 `그리드 설정 · 드래그해서 이동` 부분을 끌어 원하는 위치로 옮길 수 있습니다.
- `새 영역 선택` 버튼을 누르면 기존 그리드와 음영을 제거하고 새로운 영역을 선택합니다.
- `그리드 닫기` 버튼으로 현재 그리드를 닫을 수 있습니다.

---

## 3. 기본 사용 순서

1. `GridOverlay.exe`를 실행합니다.
2. `F8`을 누릅니다.
3. 그리드를 표시할 그래프 영역을 마우스로 드래그합니다.
4. 자동 생성된 그리드를 휠 또는 설정창으로 조절합니다.
5. 방향키로 그리드 시작 위치를 그래프 눈금에 맞춥니다.
6. 작업을 마치면 `Esc` 또는 설정창의 `그리드 닫기`를 사용합니다.

---

## 4. 조작법

| 입력 | 기능 |
| --- | --- |
| `F8` | 새 영역 선택 시작 |
| 일반 휠 | X/Y 간격을 함께 2px 조절 |
| `Ctrl + 휠` | X 간격만 0.2px 조절 |
| `Shift + 휠` | Y 간격만 0.2px 조절 |
| `Ctrl + Shift + 휠` | X/Y 간격을 함께 0.2px 조절 |
| 방향키 | 그리드 시작 위치 1px 이동 |
| `Shift + 방향키` | 그리드 시작 위치 5px 이동 |
| `Enter` | 그리드 잠금 및 클릭 통과 |
| `Ctrl + Shift + E` | 편집/잠금 전환 |
| `Esc` | 선택 취소 또는 편집 중인 그리드 닫기 |

> 잠금 상태에서는 입력이 뒤쪽 프로그램으로 전달되므로 `Esc`가 그리드에 전달되지 않습니다. 이 경우 `Ctrl + Shift + E`로 편집 상태로 돌아온 뒤 닫습니다.

---

## 5. 동작 상태

### 대기

- 프로그램이 백그라운드에서 실행됩니다.
- `F8` 입력을 기다립니다.

### 영역 선택

- 화면을 반투명하게 표시합니다.
- 사용자가 드래그한 영역과 픽셀 크기를 보여줍니다.

### 편집

- 선택 영역에 그리드를 표시합니다.
- 휠, 방향키 및 설정창으로 그리드를 조절합니다.

### 잠금

- 그리드는 유지하면서 마우스 입력을 뒤쪽 프로그램으로 통과시킵니다.
- `Ctrl + Shift + E`로 편집 상태로 돌아갈 수 있습니다.

---

## 6. 설정 저장

- 마지막 선 색상, 투명도, 주선 설정 등을 JSON 파일로 저장합니다.
- 사용자별 저장 위치: `%APPDATA%\GridOverlay`
- 설정 파일이 없거나 읽을 수 없으면 기본 설정으로 실행합니다.

## 7. 배포 방식

### Windows 무설치 버전

- 배포 파일: `GridOverlay-Windows-v0.1.zip`
- Python이 설치되지 않은 Windows PC에서도 사용할 수 있습니다.
- 사용자는 ZIP을 압축 해제한 뒤 `GridOverlay.exe`를 실행합니다.
- `GridOverlay.exe`와 `_internal` 폴더를 같은 위치에 유지해야 합니다.
- 설치 프로그램이나 관리자 권한은 필요하지 않습니다.

### 배포 시 참고사항

- 새로 만든 서명되지 않은 프로그램이므로 Windows SmartScreen에서 `Windows의 PC 보호` 경고가 표시될 수 있습니다.
- 회사 내부 배포 시 사내 IT 또는 보안 담당자의 검토와 허용 목록 등록이 필요할 수 있습니다.
- ZIP에는 사용법과 제3자 라이선스 안내 파일을 포함합니다.

---

## 8. 기술 구성

- 언어: Python 3.12
- GUI: PySide6 / Qt 6
- 전역 단축키: Windows API `RegisterHotKey`
- 설정: JSON
- 로그: Python `logging`
- 배포: PyInstaller `onedir`
- 실행 환경: Windows 10/11

화면 캡처, OCR, 네트워크 통신 및 전역 마우스 훅은 사용하지 않습니다.

---

## 9. v0 제한사항

- Windows만 지원합니다.
- 한 번에 하나의 그리드만 표시할 수 있습니다.
- 실제 그래프 수치 자동 계산 기능은 없습니다.
- 수평 기준선과 십자선 기능은 아직 없습니다.
- 선택 영역 균등 분할 모드는 아직 없습니다.
- 사용자 단축키 변경 기능은 없습니다.
- 프로그램 코드 서명은 적용되지 않았습니다.
- 다양한 듀얼 모니터 및 혼합 DPI 환경에 대한 추가 테스트가 필요합니다.
