# Grid Overlay MVP

Windows 화면 위에 선택 영역만 그리드를 표시하는 오버레이 도구입니다. 화면을 캡처하거나 외부로 전송하지 않습니다.

그래프와 이미지의 간격·정렬을 화면 위에서 빠르게 확인하기 위해 만든 Windows 데스크톱 유틸리티입니다. 영역을 선택한 뒤 격자 간격과 시작 위치를 세밀하게 조절할 수 있으며, 잠금 상태에서는 클릭을 아래 창으로 통과시켜 원래 작업을 계속할 수 있습니다.

## 핵심 기능

- 선택한 영역에만 투명 격자와 주선 표시
- 마우스 휠을 이용한 X/Y 간격 조절
- 키보드를 이용한 0.2px 단위 간격 및 1px 단위 위치 보정
- 잠금 상태의 마우스 클릭 통과
- Windows 전역 단축키를 이용한 영역 선택 및 편집 전환
- 사용자별 설정 저장과 로컬 로그 기록

## 기술 스택

- Python 3.12+
- PySide6 / Qt
- Windows API (`RegisterHotKey`)
- PyInstaller

## 설계 기준

- 화면을 캡처하거나 네트워크로 전송하지 않습니다.
- 전역 마우스 훅 없이 Qt 이벤트와 Windows 전역 단축키만 사용합니다.
- 설정과 로그는 프로젝트 폴더가 아닌 `%APPDATA%\GridOverlay`에 저장합니다.
- 잠금과 편집 상태를 분리해 오버레이 아래의 프로그램을 계속 조작할 수 있도록 했습니다.

## 설치 및 실행

1. Python 3.12 이상을 설치하면서 `Add Python to PATH`를 선택합니다.
2. `setup.bat`을 한 번 실행합니다.
3. 이후 `run.bat`으로 실행합니다.
4. 작업 표시줄 알림 영역에 프로그램이 상주합니다.

## 다른 사람에게 공유하기

`build_release.bat`을 실행하면 PyInstaller를 설치하고 Python이 없는 Windows PC에서도
실행할 수 있는 `GridOverlay-Windows-v0.1.zip`을 생성합니다. ZIP을 전달받은 사용자는
압축을 푼 뒤 `GridOverlay.exe`를 실행하면 됩니다. 설정과 로그는 사용자별
`%APPDATA%\GridOverlay`에 저장됩니다.

## 프로젝트 구조

| 파일 | 역할 |
|---|---|
| `main.py` | 영역 선택, 오버레이 렌더링, 설정 UI, 단축키 처리 |
| `requirements.txt` | Python 의존성 |
| `setup.bat` | 가상환경과 실행 환경 설정 |
| `run.bat` | 로컬 실행 |
| `build_release.bat` | PyInstaller 기반 Windows 배포 파일 생성 |

## 개발 환경에서 확인하기

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python main.py
```

## 조작법

| 입력 | 기능 |
|---|---|
| `F8` | 새 그리드 영역 선택 |
| 마우스 휠 | X/Y 그리드 간격을 2px씩 조절 |
| `Ctrl + 휠` | X 간격만 0.2px씩 정밀 조절 |
| `Shift + 휠` | Y 간격만 0.2px씩 정밀 조절 |
| `Ctrl + Shift + 휠` | X/Y 간격을 0.2px씩 정밀 조절 |
| 방향키 | 그리드 시작 위치를 1px 이동 |
| `Shift + 방향키` | 시작 위치를 5px 이동 |
| `Enter` | 편집을 잠그고 클릭 통과 |
| `Esc` | 그리드와 주변 음영 닫기 |
| `Ctrl + Shift + E` | 편집/잠금 전환 |

편집 화면의 설정 패널에서 X/Y 간격을 직접 입력하고 선 색상, 투명도,
주선 표시 여부와 주선 간격을 변경할 수 있습니다. 주선은 기본적으로 5줄마다
더 굵고 진하게 표시됩니다.

선택 후 주변 화면에는 약 20%의 음영이 남으며 축, 범례, 텍스트는 계속 볼 수 있습니다. 선택 영역 안에는 음영이 적용되지 않습니다.

새 영역을 선택하면 가로와 세로가 각각 약 10칸이 되도록 초기 간격이 자동으로
생성됩니다. 설정 패널은 그래프를 가리지 않도록 선택 영역 바깥쪽 여백에 나타납니다.
마우스를 선택 영역 위에 올리고 휠을 움직이면 별도 클릭 없이 간격이 조절됩니다.
PC의 투명 창 처리 방식 때문에 선택 영역 위 휠이 동작하지 않는 경우에는 설정창의
`여기에 마우스를 올리고 휠` 영역에서 스크롤하면 동일하게 간격을 조절할 수 있습니다.
이 기능은 프로그램 창 내부의 일반 Qt 이벤트만 사용하며 전역 마우스 훅은 사용하지 않습니다.
설정창 상단의 `그리드 설정 · 드래그해서 이동` 부분을 마우스로 끌면 설정창을
원하는 화면 위치로 이동할 수 있습니다. X축 간격은 Windows에서 안정적으로 인식되는
`Ctrl+휠`로 조절합니다.

설정창의 `새 영역 선택` 버튼을 누르면 기존 그리드와 주변 음영이 즉시 사라지고
새 영역을 드래그할 수 있습니다. 새 선택 화면에서 Esc를 누르면 선택만 취소되고
이전 그리드는 복원되지 않습니다. 선택 영역에는 별도의 최소 크기 제한이 없습니다.

## 현재 MVP 범위

- Windows 전용
- 한 번에 오버레이 한 개
- 픽셀 간격 방식의 전체 그리드
- F8 전역 단축키
- 선택 후 주변 음영 유지
- 마우스 휠 간격 조절
- 방향키 위치 정렬
- 잠금 상태에서 마우스 클릭 통과

수평 기준선, 십자선, 균등 분할, 영역 크기 재조정 UI는 다음 단계 기능입니다.
